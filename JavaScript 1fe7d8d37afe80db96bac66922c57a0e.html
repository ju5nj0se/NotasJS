SPA

Un SPA (single page aplicaction) es una sitio web de una sola pagina en donde las diferentes secciones/modules de la sitio actualiza el contenido dinamicamente sin recargar la pagina mediante que el usuario va interactuando con la pagina

¿Que define un SPA?

  • Una sola carga de pagina → El nevagador carga una sola vez el index.html luego no hay cargas completas
  • Navegacion sin recarga → Cambias entre vistas o rutas con ayuda de JS
  • Despues de la primera carga, solo se intercambian datos con el servidor

Organizacion del proyecto (usualmente para proyectos SPA)

Carpeta assets → Van todos los recursos visuales de la pagina (styles, imagenes)

Views o components → Para los componentes de la pagina

Concatenacion

  • La forma mas comun es usando + .
  • Hay otra forma que es usando la funcion concat().

Ejemplo

let cadena1 = "Hola";
let cadena2 = "Mundo";
let resultado = cadena1.concat(" ", cadena2); // Resultado: "Hola Mundo"
console.log(resultado);
