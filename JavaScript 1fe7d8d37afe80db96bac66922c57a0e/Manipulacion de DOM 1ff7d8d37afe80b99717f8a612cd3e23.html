Lista de metodos para seleccionar elementos en el DOM

Metodos de seleccion de elementos individuales

  1. Mediante su id → get.elementById("id") .
  1. General → querySelector().
    1. Este metodo selecciona el id, clase o name que le especifiquemos en los parentesis. Ejemplo
      • querySelector('p') → Selecciona el primer parrafo.
      • querySelector('.claseCSS') → Selecciona la primera clase que encuentre con ese nombre.
      • querySelector('#id') →Seleciona el primer id que encuentre.
      • querySelector('input[name="name"]') → Selecciona un input con el artributo name=”name” que encuentre.
      • Tip

        Podemos selecionar un elemento con especifidad asi:

        querySelector('div #id')

    Cual es mejor? Cual deberia utilizar?

    querySelector: Es el mas moderno.

    id: Es mas directo, utilizalo para elementos unicos y te garantiza unicidad.

    class: Permite seleccionar uno o varios elementos o componentes reutilizables

    name: Principalmente se utiliza para elementos de formulario (inputs, radios, checkboxes). Ademas de que semanticamente es correcto para formularios

Metodos de seleccion de varios elementos

Es importante resaltar que estos metodos devuelven colecciones de elementos. Puede ser una coleccion de tipo NodeList (estatica o viva) o de tipo HTMLCollection que siempre es viva.

¿Que significa que una coleccion sea estatica o viva?

Estatica → Una coleccion estatica captura instantaneamente los elementos del DOM. Si se modifica este despues de haber creado la lista, en la lista no se vera reflejado el cambio.

Viva → Una coleccion viva se actualiza dinamicamente para reflejar los cambios en el DOM. Si hay cambios en este despues de haber creado la lista, en la lista se vera reflejada.

  1. getElementsByClassName('claseCSS') → Selecciona todos los elementos con esa clase - Devuelve un HTMLCollection
  1. getElementsByTagName('p') → Selecciona todas los elementos con etiqueta p - Devuelve un HTMLCollection
  1. getElementsByName('name') → Selecciona todos los elementos que para el atributo name su valor es name - Devuelve un NodeList en vivo
  1. querySelectorAll('clasesCSS') → Selecciona todos los elementos que tengan el nombre del selector CSS especificado - Devuelve un NodeList estatico

Propiedades de navegación del DOM (relacionadas con selectores)

Aunque no son selectores en sí mismos, estas propiedades permiten acceder a elementos relacionados una vez que ya tienes una referencia a un elemento.

  • elemento.children: Devuelve un HTMLCollection de los elementos hijos (sin nodos de texto).
  • elemento.firstElementChild: Devuelve el primer elemento hijo.
  • elemento.lastElementChild: Devuelve el último elemento hijo.
  • elemento.nextElementSibling: Devuelve el siguiente elemento hermano (ignorando nodos de texto).
  • elemento.previousElementSibling: Devuelve el elemento hermano anterior (ignorando nodos de texto).
  • elemento.parentElement: Devuelve el elemento padre.

Metodos para modificar el contenido HTML

innerHTML → Permite cambiar el contenido HTML dentro de un documento

outerHTML → Similar al innerHTML pero cambiar el elementos por completo incluyendo a la etiqueta HTML del propio elemento

Modificacion de estilos

style nos permite modificar directamente los estilos de un elemento através de la propiedad style.

Modificacion de clases

Mediante classList permite modificar las clases de un elemento

  • classList.add() → Añade una clase a un elemento-
  • classList.remove() → Elimina una clase de un elemento
  • classList.toggle() → Alterna una clase; si esta presente, la elimina sino la añade
  • classList.contains() → Verifica si un elemento contiene una clase

Crear y agregar elementos

Podemos crear nuevos elementos HTML usando el metodo createElement() al igual que podemos agregarlos al DOM con appendChild() o insetBefore()

  • appendChild() → Tener en cuenta que appendChild los agrega al final del elemento
  • insertBefore() → Este metodo inserta un elemento hermano de un nodo padre antes

Clonar elementos

Con cloneNode() podemos clonar elementos HTML.

Acepta un argumento booleano:

true → En caso de quieras hacer una clonacion profunda, es decir clona el nodo y sus nodos hijos

false → Solo clona el nodo, si no se pone el argumento se asume false

Reemplazar elementos

Con replaceChild() permite reemplazar un nodo que existe por uno nuevo

Eliminar elementos

Para eliminar elementos utilizamos remove().

Recorrido del DOM

  • Acceder al nodo padre → parentNode
  • Accder a los nodos hijos (Incluyendo nodos de texto y comentarios) → childNodes
  • Acceder solo a los nodos hijos(solo devuelve elementos html) → children

Acceder a los elementos hermanos de un nodo

  • nextSibling → Retorna el siguiente nodo hermano
  • nextElementSibling → Retorna el siguiente nodo hermano que sea HTML
  • previousSibling y previousElementsSibling: → Lo mismo anterior solo que retorna el hermano anterior
