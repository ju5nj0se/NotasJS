Tipos de datos

Primitivos

number

Para declarar datos enteros o decimales JS los toma como number (se usa el punto para diferencia entre la parte entera y la parte decimal).

infinity → Representa numeros infinitos, se utiliza para representar un numero mayor que cualquier otro numero representable en el sistema.

NaN → Aparece cuando un operacion no produce un resultado numerico

BigInt → Lo utilizamos para representar numeros grandes que escapan de la zona segura. La syntaxis para decif un numero va a ser bigInt es poniendole una n al final.

Tener en cuenta que bigInt solo funciona con numeros enteros

string

Se le conoce como cadena de caracteres a una variable que guarda texto. En JS definimos una variable de tipo texto usando las comillas

escape sequences

template strings

Con las comillas invertidas podemos hacer cadenas dinamicas insertando directamente variables sin tener que concatenar manualmente.

boolean

Este tipo de dato solo acepta dos valores verdadero (true) o falso (false).

undefined

Este tipo de dato devuelve undefined cuando una variable fue declarada pero inicializada.

null

Es la forma de representar que una variable fue inicializada con un valor vacio.

Symbol

Lo utilizamos para representar identificadores unicos e inmutables. Se utiliza para crear propiedades de objetos que no pueden ser accedidas o modificadas por otros codigos.

bigInt

Almacena un numero grande

Datos complejos

En JS las estructuras de datos son fundamentales para organizar y almacenar informacion, las mas usadas son las arrays y los objetos.

arrays - listas- vectores

Son una lista ordenada de datos.

const example = ["value1", "value2"]

Metodos para los arrays

findLast → Devuelve el ultimo elemento de una lista que cumpla con una condicion.

toReversed → Crea una lista en orden inverso con los elementos de otra lista.

Objetos

Los objetos son colecciones de pares (clave:valor).

const object = {

“property1” = “value”,

"property2" = "value2"

}
