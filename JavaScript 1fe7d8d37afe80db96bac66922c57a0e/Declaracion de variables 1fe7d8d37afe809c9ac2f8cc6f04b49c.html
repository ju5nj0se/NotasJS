Declaracion de variables

Las variables son espacios donde puedes almacenar informacion. En JavaScript una variable se declara con let , const y var .

let

Es la manera mas moderna y se usa cuando quieres que el contenido de la variable cambie

Ej → let texto = "Esta variable guarda un texto"

const

La utilizamos cuando queremos que un valor no cambie, que permanezca igual.

Ej → const numero = 123

Buenas practicas para la declaracion de variables

  • Evitar espacios para nombrar variables.
  • Usar camelCase.
  • JS es sensible a las mayúsculas.
  • Siempre usar la palabra reservada para declarar variables.
  • Nombres descriptivos a los nombres de las variables.

Trucazo

Puedes declarar varias varibles a la vez asi.

let name = "Juan", edad = 17, estadoCivil = "Soltero"
